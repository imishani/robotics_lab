### Automatic Hand-to-Eye Calibration for Dobot Magician and Realsense D415

***

Requirement:

Hardware:

- Dobot Magician

- RealSense D415


Software:
- Dobot Magician SDK
- Realsense SDK 2.0
- OpenCV with Aruco module
- Python3

For more detail, please refer to: https://www.three.ml/2018/08/hand-eye-calibration/
